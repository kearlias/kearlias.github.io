Tabletop RPG Terrain Board

The above two images are the beginning of a prototype I build for a modular tabletop RPG tarrain board.

The above three images are the construction of the boards top surface. This is where world components would be placed to build a tabletop RPG world.

The last three images are the final assembly and testing of the prototype. This is the first prototype for a modular tabletop RPG terrain board. My goal with this project was to put together a small box with contacts on the top so that when another circuit was connected to it it would complete the circuit and activate whatever was attached to the top. The next prototype is planed to be activated witha switch so that the components on top dont have to be removed to turn off the box.

Board in a box

In this project we were given a sheet of metal and had to build a box to specified demensions. On one side we had to drill out a few holes and rived it together and on the other we had to spot weld the sides together. Then we had to take a proto board and solder in a number of components then mount the board into the box.

4 bit adder circuit

This project was to build and understand how adder chips work in circuits. We had to take two four-bit binary inputs off a 5Vdc power supply, run them through a 74LS283 chip, and get an output with a carry value that was the added total of the inputs.
